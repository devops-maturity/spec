## 企业版功能

### 🔒 私有 / 本地部署

在您自己的基础设施上完整运行 DevOps 成熟度评估系统。将敏感评估数据保留在您的服务器上，满足数据驻留要求，并与内部网络无缝集成。

### 👥 团队与组织管理

按团队、项目或业务部门组织评估。通过细粒度的基于角色的访问控制（RBAC）分配角色和权限，确保合适的人员拥有合适的访问权限。

### 📊 高级分析与报告

通过跨团队趋势分析、历史评分追踪、团队基准对比，以及突出薄弱实践领域的分类拆解，获得更深入的洞察。将评估数据导出为 JSON 或 CSV，接入您自己的 BI 和报表工具。

### 🔌 REST API 访问

通过完整文档化的 REST API 将 DevOps 成熟度评估集成到您现有的 DevOps 工具链中。在 CI/CD 流水线中自动化评分收集和报告。

### 🛡️ 审计日志

维护所有评估活动、配置更改和用户操作的完整记录，用于合规和安全审计目的。

### 🤝 专属支持与 SLA

获得 DevOps 成熟度团队的优先支持、有保障的响应时间以及专属客户成功经理的服务。

## 规划中的功能

以下能力计划在后续企业版发布中推出，目前尚未正式提供。如果其中某项对您的评估很重要，请[联系我们](#开始使用企业版)，我们会同步最新的时间表。

### 🔑 单点登录（SSO）

通过 SAML 2.0、LDAP 或 OIDC 与您组织的身份提供商集成，简化访问管理并强制执行集中式身份验证策略。

### 🎨 自定义品牌

使用您组织的徽标、颜色和域名对界面进行白标定制，提供无缝的内部用户体验。

### 📋 合规报告

基于企业版审计日志，提供可导出的、映射到 SOC 2 和 ISO 27001 等常见合规框架的报告。

## 企业版交付方式

DevOps 成熟度企业版以**自托管（self-host）**应用的形式交付，您可以使用 Docker Compose 在自己的基础设施上运行。

- **自助试用** —— 每个新部署都内置 14 天试用，无需 license key 即可评估。
- **授权档位** —— 选择适合您组织的档位：

| 档位 | 用户数 | 团队数 | 每月评估数 |
|------|:------:|:------:|:----------:|
| Trial | 10 | 3 | 50 |
| Starter | 100 | 20 | 500 |
| Enterprise | 10,000 | 500 | 100,000 |

- **商业打包** —— 一次性的部署/接入费用、年度支持与升级订阅，以及可选的高级支持 / SLA。
